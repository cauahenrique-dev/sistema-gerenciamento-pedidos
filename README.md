# 🛒 Sistema de Gerenciamento de Pedidos
Sistema em Java para gerenciamento de clientes, produtos e pedidos, com persistência em banco de dados MySQL via JDBC.

## 🚀 Funcionalidades

* CRUD de clientes
* CRUD de produtos
* CRUD de pedidos

## 🛠️ Tecnologias

* Java
* MySQL
* JDBC
* POO
* DAO

## ▶️ Execução

1. Configurar o banco de dados MySQL
2. Ajustar a conexão no arquivo `Conexao.java`
3. Executar a classe principal

## 📌 Objetivo

Projeto desenvolvido para prática de integração com banco de dados e aplicação dos conceitos de POO e DAO.
